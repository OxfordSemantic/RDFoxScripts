# RDFoxScripts
Example script to use with RDFox. To execute the script in RDFox.

1-Obtain an evaluation license for RDFox at https://www.oxfordsemantic.tech/request-eval.
2-Put the license you will receive via email into the same folder as the RDFox executable.
3-From the git project folder, run:
  <path_to_RDFox>/RDFox sandbox . startup.script
4-Examine the script to pick up tips on how to write scripts to make your life easier.
5-Examine the data, queries, rules, and scripts in the respective folders to learn about graph databases if you are not familiar with the space.
6-Look at BONUS_access_control_notes.txt to find out the basics of named-graph based security in RDFox.

Under Data/ there are two Turtle files, one TriG file, and one csv.
Under Queries/ there are four queries in SPARQL.
Under Rules/ there are four rules in Datalog.
Under Scripts/ there are two scripts, one to specify prefixes and one to connect RDFox to the csv.
